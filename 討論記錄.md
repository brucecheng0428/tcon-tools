## 2026-06-01 — LA/Tcon tab 匯入檔案後於 header 顯示檔名

**主題**：使用者匯入檔案後，要在 header 顯示匯入的檔名。

**討論摘要**：
- 桌面版：放在「波形產生器標題」與「右側三個 tab」之間的中間空白（藍底膠囊、置中、📄 圖示、長檔名以 ellipsis 截斷）。
- 手機版（≤480px，中間空白消失）：放在標題正下方，整列置中。
- LA（.kvdat）＋ Tcon（.txt／.json）兩個匯入處理器都要接上。
- 行為：匯入成功才顯示、各 tab 記各自檔名、切 tab 更新顯示、未匯入或載入 preset 時不顯示。

**決定／實作**：
- 新增 `#wfg-import-filename` 元件 ＋ 對應 CSS。
- 新增 `_wfgImportedNames{tcon, la}` 狀態，搭配 `wfgSetImportedFileName` / `wfgUpdateImportedFileNameDisplay`。
- 兩個匯入處理器各加一行設定檔名。
- 以 Chrome MCP 實際匯入真實檔驗證（桌面／手機／未匯入／切 tab）通過。

**狀態**：✅ 已完成上線。版號 v2.97.406，commit bd9cf45。

## 2026-06-01 — LA tab 兩個滑鼠互動 bug 修正（並查 Tcon tab）

**主題**：修正 LA tab 兩個滑鼠互動 bug，並確認 Tcon tab 是否有同樣問題。

**討論摘要**：
- Bug1：垂直虛線時間游標會越界到左側通道名稱區（波形繪圖區外）。
- Bug2：即時測量的水平白色箭頭不跟著滑鼠左右移動，要滾動滾輪才會動。

**根因／決定**：
- Bug1：crosshair 只判 `channel ≥ 0`，沒判滑鼠 x 是否在繪圖區內 → 新增 `wfgLaHoverInPlot()` 邊界判斷，越界即不畫。
- Bug2：箭頭只在 zoom 路徑重算，mousemove 沒重算 → 抽出重算函式，並於 mousemove overlay 呼叫。
- Tcon tab 經查無此兩問題（獨立實作，本來就有邊界判斷 ＋ mousemove 重算），未動 Tcon。
- 以 Chrome MCP 用 Bruce 提供的條件實測通過。

**狀態**：✅ 已完成上線。版號 v2.97.405，commit 5eca1e2。

## 2026-05-31 — tcon-tools 加密碼登入（多帳號＋登入歷程）

**主題**：為 tcon-tools 加上密碼登入，達到「真保護」並記錄登入歷程。

**討論摘要**：
- 要讓「未登入者拿不到工具內容」（真保護，不是前端遮罩可繞過的假保護）。
- 需自訂 2~3 組帳號密碼。
- 要記錄登入歷程（登入 log）。
- 大陸同事（瑞鼎，有 VPN）要能正常使用。
- 個人非營利用途，要求免費、好維護。

**決定（已定案方向）**：
- 採 **Cloudflare Worker + D1**（免費 serverless）。
- Worker 當「驗證閘 + 內容輸出」：驗證通過才吐出工具內容。
- 使用 **HMAC cookie** 維持登入狀態。
- **D1** 資料庫存登入 log。
- 部署方式從 GitHub Pages 改為部署 Worker。

**被排除的方案**：
- GitHub Pages 前端密碼閘 → 可繞過，非真保護。
- Cloudflare Access → 不支援自訂帳密。
- Vercel → 費用偏貴／灰色地帶。
- Firebase → 大陸封鎖 Google，同事無法用。
- VPS／NAS → 維護成本太重。

**狀態**：🟡 已討論定案、尚未實作（Bruce：先討論，之後有空再做）。
